📱 Android Installation

  1. Tap Download APK for Android
  2. Open the downloaded file from notifications
  3. If prompted, tap Settings and enable Install from this source
  4. Go back and tap Install
  5. Open the app and grant the requested permissions
  6. The app will automatically connect to the MDM server

 iOS Information

  1. iOS support is coming soon
  2. iOS requires Apple MDM enrollment profiles which need an Apple Developer account and MDM certificate
  3. When available, iOS devices will be enrolled via a configuration profile link
  4. Contact your IT administrator for iOS enrollment status
